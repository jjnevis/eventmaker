What is Event Maker?

Calendar event entry done right. Event Maker is a highly optimised tool for entering events into your calendar. It is NOT another calendar app, it's purpose is to do one thing very well - compose calendar events.

Features:

  1. Simple touch selection of date and time
  2. Dynamically updated event summary
  3. Add multiple alerts with a touch
  4. Tagging feature - great for searching your calendar
  5. Simple repeat events
  6. Intelligent event title pick list
  7. Logging feature - great for recording an event log
  8. Logical auto separation of event type - Log, All day, Timed appointment or Reminder (all color coded)
  9. Reset button
  10. Fully featured 'old school' form
  11. Quick calendar check - flips to calendar app on chosen date
  12. Lightning fast

Once you've used Event Maker, you won't go back.
